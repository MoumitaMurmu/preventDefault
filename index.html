Event Object

When an event occurs in HTML, the event belongs to a certain event object, like a mouse click event belongs to the MouseEvent object.

All event objects are based on the Event Object. event objects inherit the Event Object's properties and methods.

Object : Event

Event Objects : 1.DragEvent => ondrag, ondragend, ondragenter, ondragleave, ondragover, ondragstart, ondrop 2.KeyboardEvent => onkeydown, onkeypress, onkeyup 3.MouseEvent => onclick, oncontextmenu, ondblclick,onmousedown, onmouseenter, onmouseleave, onmousemove, onmouseout, onmouseover, onmouseup
